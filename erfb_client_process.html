send_event/2

send_event(Client::fsmref(), Event::client_event()) -> ok

Sends an client event to the server. The event may be one of the following ones (each one represented with a corresponding record):

  • client_connected
  • listener_disconnected
  • client_disconnected
  • set_pixel_format
  • set_encodings
  • update_request
  • key
  • pointer
  • client_cut_text
For a description of the events, check the RFB Protocol Definition
